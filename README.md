# Snake Palaclone

Clon del clásico **Snake** hecho en JavaScript puro para navegador, optimizado para desktop y celular.

## 🎮 Jugar online

👉 https://paladroncillo.github.io/Snake-palaclone/

> Si no abre todavía, activa GitHub Pages en:
> **Settings → Pages → Source: GitHub Actions**

## Controles

### Teclado
- **Flechas** o **WASD**: mover
- **P**: pausar / continuar
- **R**: reiniciar

### Celular
Botones táctiles flotantes:
- ▲ arriba
- ◀ izquierda
- ▼ abajo
- ▶ derecha
- ⏯ pausa

## Características

- Puntaje en tiempo real
- Récord guardado en `localStorage`
- Velocidad progresiva al comer comida
- Pantalla de pausa y game over
- UI responsive para móvil
- Workflow de GitHub Actions para desplegar en GitHub Pages en cada push a `main`

## Ejecutar localmente

Abre `index.html` en tu navegador.

Opcional con servidor local:

```bash
python3 -m http.server 8080
```

Luego entra a:

`http://localhost:8080`
